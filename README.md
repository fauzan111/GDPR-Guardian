# GDPR-Guardian
In the Italian/European market, GDPR Compliance and Privacy by Design are mandatory. This service automatically detects and redacts sensitive data like the Codice Fiscale using custom Presidio recognizers and Spacy's Italian model, ensuring data safety before it hits the database.

This project demonstrates:

MLOps: Full end-to-end containerization with Docker.

Data Engineering: Building an API gateway for data governance.

Full-Stack Data Science: From custom logic to deployment.
